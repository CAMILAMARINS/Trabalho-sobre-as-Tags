Trabalho sobre as Tags

1_Tag header:

são elementos HTML usados para estruturar o conteúdo de uma página da web. Essas tags são usadas para indicar os títulos e subtítulos de uma página, tornando mais fácil para os mecanismos de busca e usuários entenderem a hierarquia e o contexto do conteúdo.


Tag footer:

elemento footer marca a área inferior, normalmente conhecida como rodapé, do conteúdo geral do documento ou do conteúdo de uma seção específica a qual ele está subordinado.


Tag main:

main serve para indicar o conteúdo principal ou a área principal do site. Faz mais sentido a section (ou várias) dentro de uma main e não o contrário. Mas se você usar um main dentro de uma section você está explicitando que aquele pedaço é a área mais importante e principal do site.


Tag section:

é utilizada para marcar as seções de conteúdo de uma página. Com Esse elemento agrupamos de forma lógica nosso conteúdo, separando a informação em áreas diferentes. O principal objetivo é retirar essa responsabilidade das divs.


Tag anticle:

destina-se a marcar um conteúdo que se constitua em um componente autossuficiente em uma página, aplicação ou site e que possa ser distribuido ou reusado de forma independente (isolada), como por exemplo via sindicalização.


Tag aside:

representa uma seção de uma página que consiste de conteúdo que é tangencialmente relacionado ao conteúdo do seu entorno, que poderia ser considerado separado do conteúdo. Essas seções são, muitas vezes, representadas como barras laterais.


2_ Tag fieldset:

é usado dentro de um form para agrupar elementos do form que são relacionados. Por padrão ele mostra visualmente esses elementos agrupados dentro de uma caixa com um título. é usado para agrupar elementos, assim como labels ( ), dentro de um formulário web.
